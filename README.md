# BelgradeATC

An empty ASP.NET Core Web API project.

## Getting Started

### Prerequisites

- .NET 10.0 or later
- Visual Studio Code or Visual Studio 2022

### Running the Project

To run the API locally:

```bash
dotnet run
```

The API will be available at `https://localhost:7000` (or the port specified in `launchSettings.json`).

### Building the Project

To build the project:

```bash
dotnet build
```

### Project Structure

- **Program.cs** - Application entry point and service configuration
- **BelgradeATC.csproj** - Project file with dependencies and configuration
- **Properties/** - Contains launchSettings.json for development configuration
- **appsettings.json** - Application settings
- **appsettings.Development.json** - Development-specific settings
